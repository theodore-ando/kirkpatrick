proj.independentset module¶

planar_independent_set(graph: networkx.classes.graph.Graph, black_list: typing.List = [], degree_lim=8) → typing.Set[polygons.Point][source]¶

Assuming the graph is planar, then computes an independent set of size at least n/18 in which every node has degree at most 8. O(n) time.

(Note: assume the graph is planar because it is computationally expensive to check)

Parameters:
  • graph – networkx Graph (assumed planar)
  • black_list – list of nodes that should not be in the independent set
  • degree_lim – nodes in independent set have degree at most this
Returns:

set of graph nodes
