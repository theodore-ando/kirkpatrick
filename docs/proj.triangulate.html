proj.triangulate module¶

triangulate(poly: polygons.Polygon, hole: polygons.Polygon = None) → typing.List[polygons.Triangle][source]¶

Triangulates a polygon, potentially with a hole in it. Uses naive O(n^2) ear-clipping method.

Parameters:
  • poly – polygon to be triangulated
  • hole – hole in the polygon
Returns:

list of triangles making up the polygon
